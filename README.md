# Task Forge

Aplicação de lista de tarefas gamificada construída com React, TypeScript, Vite e Tailwind CSS.

## Funcionalidades

- CRUD completo de tarefas
- Prioridades, categorias personalizadas e datas de vencimento
- Sistema de pontos, níveis, streaks e badges
- Estatísticas com gráficos em SVG
- Persistência de dados em Local Storage
- Modo escuro com visual SaaS moderno

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
